Инструкция по работе с приложением "Конвертер запросов Админ"

Приложение "Конвертер запросов Админ" предназначено для редактирования, добавления, удаления кода запросов и категории самих запросов.

1. Редактирование кодов запроса

Для того, чтобы изменить код запроса:

1.1. Выберите необходимый справочник, путем нажатия на значок раскрывающегося списка.

1.2. Далее выберите необходимый регион.

1.2 Введите необходимые изменения в поля текстового ввода, где БО - это Базовый орган, ТО - Территориальный орган, РО - Районный орган и кликните на кнопку "Применить". Внесённые изменения сохранятся в базе данных. Для того, чтобы внесённая информация появилась в приложении "Конвертер запросов ПФР", запустите "Конвертер запросов ПФР" и получите обновление (Подробная инструкция описана в разделе справка приложения "Конвертер запросов ПФР").

2. Добавление нового региона (Базового органа)

2.1 Для добавления нового региона в общий список необходимо на одном из справочников кликнуть правой клавишей мыши. Появится контекстное меню, в котором кликнете "Добавить элемент".

2.2. Откроется диалоговое окно "Добавление нового элемента". Убедитесь в заголовке окна, что вы добавляете именно регион. Далее введите название региона и его коды. После чего кликните на кнопку "Добавить".

2.3. Новый регион отобразится в общем списке.

3. Добавление района (Районного органа)

3.1. Кликните правой клавишей мыши по региону, в который хотите добавить район. Появится контекстное меню, в котором кликнете "Добавить элемент".

3.2. Откроется диалоговое окно "Добавление нового элемента". Убедитесь в заголовке окна, что вы добавляете именно район в регион. Далее введите название населённого пункта и его коды. После чего кликните на кнопку "Добавить".

3.3. Район отобразится в раскрывающемся списке того региона, в который вы его добавили.

4. Удаление региона (Базового органа) или района (Районного органа)

4.1. Выберите необходимый регион или район из общего списка. Кликнете по нему правой клавишей мыши. Появится контекстное меню, в котором выберите пункт "Удалить элемент".

4.2. Откроется диалоговое окно "Удаление элемента". Убедитесь, что вы удаляете то, что нужно. После чего, кликните на кнопку "Удалить".

5. Изменение название запроса

5.1. Кликнете по вкладке "Запросы". Появится список запросов. Выберите необходимы запрос из списка. В правой части окна приложения в поля ввода "Полное название запроса" и "Сокращённое название запроса" введите необходимые изменения. После чего кликните по кнопке "Применить". Изменения сохрянятся в базу данных.

6. Добавление новой категории запроса

6.1. Кликнете по вкладке "Запросы". Появится список запросов. Кликните правой клавишей мыши по элементу списка "Запросы". Появится контекстное меню, в котором выберите пункт меню "Добавить элемент".

6.2. Откроется диалоговое окно "Добавление нового запроса". В поля для текстового ввода "Полное название запроса" и "Сокращённое название запроса" введите необходимые данные. После чего кликните по кнопке "Добавить". Добавленный запрос сразу сохранится в базе данных и отобразится в списке запросов.

7. Удаление категории запроса

7.1. Кликните по вкладке "Запросы". Появится список запросов. Далее проделайте аналогичные действия, как в пунктах 4.1-4.2.

8. Работа с поиском

8.1. Во вкладке "Регионы" реализована функция поиска необходимого региона или района. Для этого начните вводить название необходимого района в поле поиска. По мере ввода символов приложение будет искать необходимый населённый пункт одновременно раскрывая список. Когда необходимый элемент появится в списке кликнете по нему, значения кодов отобразятся в правой части окна приложения.

8.2. Для сброса результата поиска и возврата в начало кликните по кнопке "Сброса результатов поиска".
